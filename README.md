# Footstep Power Generation System using Arduino

## 🔹 Overview
This project generates electrical energy from human footsteps using piezoelectric sensors and stores it for small-scale applications.

## 🔹 Components Used
- Arduino Uno
- Piezoelectric Sensors
- Bridge Rectifier
- Capacitor
- LEDs / LCD Display

## 🔹 Working Principle
When pressure is applied on the piezo sensors, mechanical energy is converted into electrical energy. The generated AC signal is converted to DC using a rectifier and stored in a capacitor. Arduino is used to monitor the output.

## 🔹 Applications
- Smart Roads
- Railway Stations
- Public Walkways
- Renewable Energy Systems

## 🔹 Output
- Generates small voltage from footsteps
- Can power LEDs or small devices
